# 运行档位与机器要求

## CPU 基线模式

- 目标：无 GPU 环境也能跑通完整链路
- 推荐机器：8 核 CPU，16GB 内存起步，32GB 更稳
- 默认模型：`moka-ai/m3e-small`
- 默认策略：如果句向量模型不可用，则回退到 `TF-IDF + sklearn` 的稀疏链路

### 适用场景

- 开发联调
- 小样本 smoke test
- 中等规模数据集的第一版问题发现

## GPU 增强模式

- 目标：在更强机器上获得更好的语义表示和聚类质量
- 推荐机器：12-16 核 CPU，32-64GB 内存，NVIDIA 12GB VRAM 起步
- 默认模型：`BAAI/bge-m3`
- 默认策略：优先启用 dense encoder、UMAP 和 HDBSCAN

### 适用场景

- 跨地区大批量数据
- 复杂 issue 语义聚类
- 更稳定的类簇召回和复用

## 当前开发机说明

当前开发机只有 2 逻辑核、约 4GB 内存、无可用 NVIDIA GPU，因此：

- 适合写代码和跑 smoke test
- 不适合作为全量问题发现的正式运行环境
- 代码中必须保留 CPU 回退逻辑
